# Design System

This file will be auto-generated when you use the `/create design system` command in Claude Code.

## How to Generate

1. Open Claude Code
2. Type: `/create design system`
3. Answer questions about:
   - Color scheme (warm/cool/neutral)
   - Tone (professional/casual/playful)
   - Component style (rounded/sharp/minimal)
4. Your custom design system will be generated here

## What Gets Generated

- Color palette
- Typography scale
- Spacing system
- Component variants
- Design tokens
- Usage examples

## Applying to Your App

Once generated, use this command:

```
Apply the design system from docs/ui/design-system.md to all pages and components
```

Claude Code will update your app's styling to match your brand.
